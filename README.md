# 📱 SMARTFIX

## Sistem Pakar Diagnosa Kerusakan Smartphone Menggunakan Metode Forward Chaining

SMARTFIX merupakan aplikasi **Sistem Pakar** berbasis **ASP.NET Core MVC** yang digunakan untuk membantu pengguna dalam mendiagnosa kerusakan pada smartphone berdasarkan gejala yang dipilih menggunakan metode **Forward Chaining**.

---

# 📌 Deskripsi

Aplikasi ini dibangun sebagai proyek mata kuliah **Sistem Pakar**.

SMARTFIX memungkinkan admin mengelola data gejala, data kerusakan, basis pengetahuan (rule), serta melakukan proses diagnosa secara otomatis menggunakan metode Forward Chaining.

# 🧠 Metode

Metode yang digunakan dalam sistem ini adalah:

**Forward Chaining**

Metode ini bekerja dengan cara mencocokkan gejala yang dipilih pengguna terhadap basis pengetahuan (rule) hingga menghasilkan diagnosis kerusakan smartphone beserta solusi yang sesuai.
