## 📖 Tentang Project

SMARTFIX merupakan aplikasi **Sistem Pakar Diagnosa Kerusakan Smartphone** yang dibangun menggunakan **ASP.NET Core MVC** dengan menerapkan metode **Forward Chaining**.

Aplikasi ini dikembangkan sebagai **Tugas Kelompok Mata Kuliah Kecerdasan Buatan** untuk membantu pengguna dalam mendiagnosa kerusakan smartphone berdasarkan gejala yang dipilih. Sistem akan mencocokkan gejala dengan basis pengetahuan (rule) sehingga menghasilkan diagnosis beserta solusi yang sesuai.

# 🎯 Tujuan

Membangun sistem pakar yang mampu membantu proses identifikasi kerusakan smartphone secara cepat, mudah, dan efisien berdasarkan pengetahuan pakar yang disimpan dalam basis pengetahuan.

# 👥 Hak Akses

## 👤 User

User dapat langsung menggunakan sistem diagnosa tanpa harus login.

Fitur yang tersedia:

- Melakukan diagnosa kerusakan smartphone
- Memilih gejala yang dialami
- Melihat hasil diagnosa
- Melihat solusi perbaikan

## 👨‍💼 Admin

Admin harus melakukan login terlebih dahulu.

Setelah berhasil login, admin dapat mengakses:

- Dashboard
- Data Gejala
- Data Kerusakan
- Basis Pengetahuan (Rule)
- Riwayat Diagnosa
- Logout

# ✨ Fitur Sistem

✅ Login Admin

✅ Dashboard

✅ Diagnosa Smartphone

✅ Data Gejala (CRUD)

✅ Data Kerusakan (CRUD)

✅ Basis Pengetahuan / Rule (CRUD)

✅ Riwayat Diagnosa

✅ Metode Forward Chaining

# 🧠 Metode yang Digunakan

Metode inferensi yang digunakan adalah:

## Forward Chaining

Forward Chaining merupakan metode penalaran yang dimulai dari fakta atau gejala yang dipilih pengguna, kemudian sistem akan mencocokkannya dengan aturan (rule) pada basis pengetahuan hingga diperoleh kesimpulan berupa jenis kerusakan smartphone beserta solusi yang sesuai.
